How it's done

We use a div for a container for the whole card, and then a couple of divs inside of that to represent each side. When the container is clicked, we toggle the classes of our "side" divs using jQuery, which applies a new CSS style to them. CSS uses transitions and transforms to rotate the card. It's important that the "side" divs have backface-visibility set to hidden in their style.
